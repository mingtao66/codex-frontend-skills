
# codex-react-skills

一套可复用的 Codex 前端开发 Skills 集合，专为 **React + TypeScript** 项目设计。

帮助 Codex 自动生成 **结构清晰、可维护、可扩展的前端代码**，特别适用于：

- SaaS 后台
- 管理系统
- Shopify App
- 数据管理平台
- 表单系统
- Dashboard
- 企业级前端项目

---

# ✨ 功能特性

本项目可帮助 Codex 自动：

- 生成结构规范的 React + TypeScript 组件
- 构建标准后台管理页面布局
- 快速生成表单页面
- 自动组织 API 调用层
- 保持 UI 结构一致性
- 自动优化响应式布局（PC / 平板 / 手机）
- 自动拆分过大的组件文件
- 优化前端性能结构
- 根据任务自动选择最合适的 skill 组合

---

# 🎯 适用场景

非常适合以下项目：

- Shopify 应用
- 数据列表系统

# 🧠 包含 Skills

| Skill | 作用 |
|------|------|
| react-typescript-pro | 生成规范的 React + TypeScript 代码结构 |
| frontend-design | 页面布局与 UI 结构设计 |
| design-system-architect | 保持 UI 风格统一 |
| api-integration-frontend | API 请求层与 hooks 组织 |
| form-builder | 创建 / 编辑 / 设置 表单生成 |
| refactor-react-component | 拆分大型 React 文件 |
| responsive-layout | 响应式布局优化 |
| frontend-performance | 前端性能优化 |
| frontend-workflow-router | 自动选择合适的 skill 组合 |

# 📦 安装方式

## 自动安装

```bash
bash install.sh
```

安装目录：

```
~/.codex
```

安装完成后重启 Codex 即可。

---

# ⚙️ 工作原理

Codex 会根据你的需求描述自动匹配最合适的 skill 组合。

示例：

```
生成一个 React + TypeScript 商品管理页面：

包含：
筛选区
数据表格
编辑弹窗
API 调用
手机端适配
代码结构清晰
```

Codex 会自动组合多个 skills 生成更接近生产环境的代码结构。
